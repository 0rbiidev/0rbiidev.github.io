Work

flotise

flotise is a free, open source X11 window manager for Linux systems written in C++.

Development began in November of 2023 as a part of my University thesis concerning the structure and functionality of Linux window managers. The result of this research is a novel concept WM leveraging X11's internal window tree structure to enable automatic tiling of several appliaction windows within a single, floating frame.

The first release target was reached and submitted in April 2024 and development is ongoing as a personal project.
